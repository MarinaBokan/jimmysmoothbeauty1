# Jimmy smooth beauty — website

Statički sajt salona lepote, spreman za hostovanje na GitHub Pages. Sve je u jednom fajlu (`index.html`) — slike, fontovi i skripte su ugrađeni, radi i offline.

## Objava na GitHub Pages

1. Napravi novi repozitorijum na GitHub-u (npr. `jimmy-smooth-beauty`).
2. Uploaduj `index.html` (i ovaj `README.md`) u repozitorijum.
3. Idi na **Settings → Pages**.
4. Pod **Branch** izaberi `main` i folder `/ (root)`, pa **Save**.
5. Nakon minut-dva sajt će biti dostupan na:
   `https://<tvoj-korisnik>.github.io/<ime-repozitorijuma>/`

## Izmene sadržaja

Fajl `index.html` je kompajlovan (spojen) — nemoj ga ručno menjati. Za izmene teksta, cena, slika ili logotipa vrati se u projekat, izmeni izvorni dizajn i ponovo ga eksportuj.
